## Screen 1: Salary Statements List

**Route:** `/salary`
**Purpose:** List all monthly salary statements with status and net payable.

### Header

- `SectionHeader`: "Salary Statements" вАФ `typography.display-xs` (20px/500)
- bg `--secondary` (#f7f7f7)

### List

- `FlatList` with `RefreshControl`
- Each item as `<SalaryStatementListItem>`:

  **Card layout:**
  - bg `--surface-card`, 16px radius, Soft Lift shadow, 16px padding
  - Left: month + year e.g., "June 2026" вАФ `typography.body-emphasis` (16px/500)
  - Status badge: colored by status
    - `PAID`: bg `--semantic-up` (#22c55e), white text
    - `PROCESSED`: bg `--primary` (#024ad8), white text
    - `PENDING`: bg `--accent-yellow` (#eab308), white text
    - `FAILED`: bg `--destructive` (#b3262b), white text
    - `HELD`: bg `--charcoal` (#3d3d3d), white text
  - Net payable: "вВє66,500" вАФ `typography.display-sm` (24px/500), `--ink`
  - Payment date (if paid): "Paid on 01/07/2026" вАФ `typography.caption-md`, `--graphite`

### States

| State         | Behaviour                                                                          |
| ------------- | ---------------------------------------------------------------------------------- |
| **Loading**   | `<SalaryStatementsListSkeleton>` вАФ 6 shimmer cards                                 |
| **Empty**     | `<EmptyScreen>` вАФ "No Statements Found", "You have not created any statements yet" |
| **Error**     | `<EmptyScreen>` with refresh                                                       |
| **Populated** | Full scrollable list                                                               |

### Navigation

- Tap card вЖТ `/salary/[id]` (detail)
- Bottom tab: Salary (active)

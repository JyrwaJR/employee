# Project Stack

## Framework
- Expo (SDK 54)
- React Native
- Expo Router (File-based routing)

## State & Data
- TanStack Query (React Query)
- Axios (HTTP client with interceptors)
- Zustand / Hook Form (State management)

## Styling
- Tailwind CSS (NativeWind v4)
- Lucide / Ionicons (@expo/vector-icons)

## Infrastructure
- EAS (Build/Submit)
- Multiple app variants based on `APP_VARIANT` env var.
